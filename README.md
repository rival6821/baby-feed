# 🍼 아기 수유량 계산기 (Baby Feeding Calculator)

> **"우리 아기를 위한 똑똑하고 건강한 수유 스케줄러"**
>
> 대한소아청소년과학회 표준 지침 및 미국소아과학회(AAP) 권고 기준을 바탕으로 설계된, 아기의 발달 상태 맞춤형 수유량 계산 및 스케줄 생성기입니다.

## ✨ 핵심 기능 (Key Features)

### 1. 👶 개인 맞춤형 영양 정보 분석
* **월령별 가이드라인 매칭**: 생후 일수(Days)를 계산하여 초기 신생아(0~2주)부터 12개월 이상까지의 적정 1회 수유량 및 빈도를 매칭합니다.
* **이른둥이(미숙아) 교정연령 지원**: 재태주수(22주~36주)를 기반으로 예정일(40주) 대비 교정연령을 자동 계산하여 정밀한 영양 발달 기준을 제공합니다.
* **체중 기반 권장 수유량 산출**: 아기 체중 입력 시 소아과 표준 공식인 `체중(kg) * 150ml`를 적용하여 1일 권장 수유량을 계산합니다.

### 2. ⚙️ 유연한 커스텀 수유 설계
* **수유 횟수 직접 설정**: 아기의 소화력이나 수유 성향에 맞게 일일 수유 횟수(1~16회)를 변경하면, 이에 맞추어 1회 수유량이 자동으로 배분됩니다.
* **하루 총 수유량 직접 설정**: 사용자가 목표 수유량을 임의로 설정(100ml~2,000ml)할 수 있으며, 이에 맞춰 1회 분량이 정밀하게 조절됩니다.
* **과잉 수유 방지 안전선**: 6개월 미만 영아의 신장 및 영아 비만을 방지하기 위해 1일 총 수유량의 상한선(1,000ml 법칙)을 고려한 경고 및 조절 가이드라인을 제공합니다.

### 3. ⏰ 스마트 수유 스케줄링 & 밤잠 압축 알고리즘
* **첫 수유(기상) 시점 기준 배분**: 하루 첫 수유 시간을 입력하면 전체 타임라인이 고르게 정렬됩니다.
* **밤잠(통잠) 모드**: 아기가 밤 동안 잘 때 수유를 건너뛰는 통잠 옵션을 설정할 수 있으며, 입력된 밤잠 시작 시점과 기상 시점 사이를 제외한 **낮 시간(활동 시간) 내에만 수유 횟수가 고르게 분배되도록 간격을 압축 조정**합니다.
* **직관적인 시간표 라벨링**: 기상/기상 직후 수유(🌅 첫 수유), 취침 직전 수유(🌙 마지막 수유), 그리고 일반 낮 수유와 야간 수유(🌙 야간)를 이모지와 함께 구분하여 보기 쉽게 출력합니다.

### 4. 🔗 공유 및 지속성
* **텍스트 복사 & 공유 기능**: 계산 결과 및 시간표를 깔끔한 템플릿(이모지 포함) 텍스트로 즉시 복사하여 육아 일기 앱, 카카오톡 등에 공유할 수 있습니다. 계산기 웹사이트의 바로가기 링크가 자동으로 포함됩니다.
* **로컬 캐싱 (LocalStorage)**: 입력했던 생년월일, 체중, 교정연령 설정, 밤잠 및 첫 수유 시각, 커스텀 옵션까지 모든 데이터가 로컬 스토리지에 캐싱되어 새로고침 후에도 자동 유지됩니다.

### 5. 📚 의학적/학술적 근거 제공
* **소아과 임상 지침 수록**: 대한소아청소년과학회(홍창의 소아과학) 및 미국소아과학회(AAP)의 수유 가이드 자료에 기반한 의학적 백그라운드 지식을 아코디언 컴포넌트를 통해 열람할 수 있도록 제공하여 서비스 신뢰도를 극대화했습니다.

## 🎨 디자인 및 UI/UX 특징

* **글래스모피즘 & 부드러운 파스텔 테마**: 시각적으로 편안함을 주는 파스텔 톤 그라데이션 백그라운드와 유리 질감(Glassmorphism)의 카드 컴포넌트를 채택했습니다.
* **동적 인터랙션 및 애니메이션**:
  * 입력 에러 시 발생하는 흔들림(Shake) 모션 및 피드백.
  * 계산하기 클릭 시 수치들이 부드럽게 증가하는 롤링 애니메이션 효과.
  * 스케줄 생성 시 타임라인 카드가 부드럽게 솟아오르는 슬라이드 애니메이션.
  * 복사 완료 시 복사하기 버튼의 체크 아이콘 전환 및 피드백 애니메이션.
* **모바일 및 태블릿 대응**: 데스크톱 화면뿐만 아니라 모바일 환경에서도 터치 제어와 입력이 최적화된 완전 반응형 레이아웃을 구현했습니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Markup**: HTML5 (웹 접근성 준수, 시맨틱 태그)
* **Styling**: Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid, Media Queries, Backdrop-filter)
* **Logic**: Vanilla JavaScript (ES6+, LocalStorage API, Clipboard API, DOM Manipulation)

## 🚀 로컬 실행 방법 (How to Run Locally)

본 프로젝트는 별도의 패키지 설치나 빌드 과정이 필요 없는 순수 정적 웹 애플리케이션입니다.

### 방법 1. 단순 실행
1. 저장소를 클론하거나 다운로드합니다.
   ```bash
   git clone https://github.com/rival6821/baby-feed.git
   ```
2. 다운로드 폴더 내의 `index.html` 파일을 더블 클릭하여 웹 브라우저에서 바로 실행합니다.

### 방법 2. Python 로컬 서버 실행 (추천)
로컬 환경에서 완전한 작동을 테스트하기 위해 로컬 HTTP 서버 실행을 권장합니다.
```bash
cd baby-feed
python3 -m http.server 8080
```
그 후 브라우저를 열고 `http://localhost:8080`에 접속합니다.
